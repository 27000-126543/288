# 智慧农贸批发市场管理系统

## 项目简介

大型综合智慧农贸批发市场管理APP，集成批发商采购、商户管理、食品安全检测、智能推荐等功能。

## 功能模块

- **批发商端**：发布求购信息、浏览商品、智能推荐采购方案、下单、物流匹配
- **商户端**：商品管理、库存管理、补货预警
- **食品安全检测**：检测报告上传、电子追溯码、不合格商品自动下架
- **费用结算**：账期赊购、信用贷款、授信额度管理
- **会员体系**：普通/银卡/金卡/钻石等级管理
- **管理员看板**：实时数据监控、交易分析
- **价格预测**：基于历史数据、天气、节假日的价格走势预测
- **报表导出**：月度交易报表导出

## 技术栈

- **前端**：React 18 + TypeScript + Ant Design + Vite
- **后端**：Node.js + Express + TypeScript + MongoDB
- **数据库**：MongoDB
- **认证**：JWT

## 快速开始

```bash
# 安装所有依赖
npm run install:all

# 启动后端开发服务器
npm run dev:backend

# 启动前端开发服务器
npm run dev:frontend
```
